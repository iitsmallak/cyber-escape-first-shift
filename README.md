# Cyber Escape: First Shift

## Overview
Cyber Escape: First Shift is an AI-enhanced cybersecurity awareness simulation that places users in the role of a SOC Analyst. The project provides an interactive experience where users investigate cyber incidents, identify threats, and make security decisions in a realistic SOC environment.

## Features
- Interactive SOC Analyst simulation
- Cyber incident investigation scenarios
- Threat identification and decision-making
- AI-powered avatars and voice generation
- Security awareness feedback

## Scenarios
The simulation includes cybersecurity incidents such as:
- Phishing attacks
- Social engineering attempts
- Fake login pages
- Public Wi-Fi risks
- USB-based threats

## How It Works
1. Users start a cybersecurity mission as a SOC Analyst.
2. They analyze incidents and identify potential threats.
3. They make security decisions during simulated attacks.
4. The system provides feedback to improve cybersecurity awareness.
